Overview

We propose a groundbreaking problem: Generative Design-for-Robot-Assembly (GDfRA). This problem involves genearting and executing a design and robot assembly plan from a natural language prompt (e.g., “giraffe”) and an image of available physical components.

Blox-Net is a powerful GDfRA system which leverages a Vision Language Model, physics simulator, and real robot to generate and reliably assemble designs resembling a specified text prompt from 3D printed blocks with no human input.

Blox-Net Design Generation

Blox-Net generates designs by sequentially prompting GPT-4o to develop a design plan and place blocks in a simulator. Then, GPT-4o receives visual and stability feedback from the simulator and makes adjustments. This process is performed 10 times in parallel and GPT-4o is used to select the design which looks the best. An automated perturbation redesign process introduces tolerances and improves stability to improve the constructability of the design.

Multi-scale supervision

To supervise language embeddings, we pre-compute an image pyramid of CLIP features for each training view. Then, each sampled ray during optimization is supervised by interpolating the CLIP embedding within this pyramid.
